# ThesisForge Demo Data

This folder contains safe, synthetic demo material for hackathon judging and local walkthroughs.

## Demo Thesis

Use `demo-thesis/` to create a project that can be reviewed by the full multi-agent workflow:

1. Create a new project with the values from `demo-thesis/project.json`.
2. Paste `demo-thesis/thesis_draft.txt` into the pasted-text document flow, or upload it as a text file.
3. Upload `demo-thesis/references.bib` as reference material.
4. Upload `demo-thesis/results.csv` as the result summary file.
5. Start a thesis review from the project review page.

The thesis scenario, measurements, institution, and dataset are synthetic. The bibliography contains public, non-private academic references that are included only to make citation and literature-review agents useful during a demo.
